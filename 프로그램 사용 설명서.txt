[Windows 환경]


1. node 및 MySQL 설치! 


2. 클라이언트 실행!


: cmd(명령창) 실행 -> Client 폴더 내로 이동 (using cd 명령어) -> 

npm install (React 라이브러리를 자동으로 코드에 맞게 설치하는 명령어) -> npm run start (클라이언트 브라우저 실행 명령어)


3. 서버 실행!


: cmd(명령창) 실행 -> Server 폴더 내로 이동 (using cd 명령어) -> 

npm install (node 모듈들을 자동으로 코드에 맞게 설치하는 명령어) -> node app.js (port 3001에서 서버 열림)


4. 브라우저의 '저장' 페이지를 통해 서버로 파일을 저장할 수 있고, 

'조회' 페이지를 통해 서버에 저장한 파일 목록을 브라우저로 불러올 수 있음!


5. '조회' 기능에서 브라우저로 불러온 파일 리스트의 원소를 클릭하면, 해당 파일 원본의 무결성을 검증함.


6. 자세한 동작 방식은 Report 폴더 내의 '최종 보고서'의 '결과' 항목에 기재
